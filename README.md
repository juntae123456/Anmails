## 📝 소개
리액트와 mysql를 이용하여 인스타그램을 클론 코딩하였습니다.

구현된 기능들은 다음과 같습니다.
- 회원가입을 통한 로그인 
- 피드나 스토리를 추가
- 피드를 올린 사용자만이 삭제 가능
- 프로필 사진 변경
- 프로필 검색 가능
- 메시지 기능
- 피드에 좋아요를 누를 수 있음
